# 📌 data-encoding-systems

A simple CRUD-based web application designed to simulate real-world data encoding workflows using PHP and MySQL. This project focuses on demonstrating structured data entry, validation, and database interaction within a basic front-end interface.


---

## 📂 Project Overview

This repository features a beginner-friendly CRUD-based system built to understand the fundamentals of PHP and MySQL while integrating HTML and CSS for front-end development. The project emphasizes the interaction between front-end and back-end database operations, demonstrating structured data processing and basic record management.

---

## 💽 Features

- Structured data entry form
- Server-side data processing using PHP
- MySQL database integration
- Basic input validation

---

## 🛠 Technologies Used

- HTML5
- CSS3
- PHP
- MySQL
- XAMPP (local development environment) 

---

## 📬 How It Works

- User input structured data through a web form
- PHP processes the form submission
- Data is validated and stored in a MySQL database
- Records are retrieved and displayed in the structured table
- Edit and Delete existing records

---

## 🌐 How to View the Project

1. Install and start XAMPP
2. Place the project folder inside the htdocs directory
3. Import the provided SQL file into phpMyAdmin
4. Start Apache and MySQL services
5. Create a Database name 'crud_db', table name 'users', and lastly column name 'id (auto increment), name, email, phone, address'
6. Open the project in a browser by typing: http://localhost/user-management-db/index.php

---

## 💡 Purpose

This project was created to practice backend fundamentals and align with data encoding and database-driven application development.
